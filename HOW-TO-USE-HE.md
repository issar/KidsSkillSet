# איך להשתמש באפליקציה – גלגל כישורים

## הפעלה במחשב (Windows)

1. **פתח שורת פקודה (Command Prompt)**  
   - לחץ על מקש Windows והקלד `cmd`  
   - או: לחץ Windows + R, הקלד `cmd` ולחץ Enter  

2. **עבור לתיקיית הפרויקט**  
   ```  
   cd "C:\Users\97252\CODE projects\KidsSkillSet"  
   ```  
   (החלף בנתיב הנכון אם הפרויקט נמצא במקום אחר.)

3. **התקנת חבילות (פעם ראשונה בלבד)**  
   ```  
   npm install  
   ```  
   חכה עד שההתקנה מסתיימת.

4. **הפעלת האפליקציה**  
   ```  
   npm run dev  
   ```  
   תופיע הודעה עם כתובת, למשל:  
   `http://localhost:5173`

5. **פתיחת האפליקציה בדפדפן**  
   - פתח את הדפדפן (Chrome, Edge וכו')  
   - בשורת הכתובת הקלד: `http://localhost:5173`  
   - לחץ Enter  

6. **עצירת השרת**  
   - חזור לחלון שורת הפקודה  
   - לחץ `Ctrl + C`  

---

## ייבוא נתונים (פעם ראשונה)

- בלחיצה על **skills.csv** בוחרים קובץ כישורים מהמחשב.  
- בלחיצה על **students.csv** בוחרים קובץ תלמידים.  
- בלחיצה על **assessments.csv** בוחרים קובץ הערכות.  
- אחרי שבחרת את שלושת הקבצים, לחץ **"ייבוא שלושת הקבצים"**.  
- ניתן להשתמש בקבצי הדוגמה מתיקיית `public/sample-data/`.  

---

## ניווט

- **לוח הבקרה** – רשימת כל הקבוצות.  
- **לחיצה על קבוצה** – נכנסים לטבלת התלמידים בקבוצה (שם, תאריך הערכה אחרון, שני כישורים נמוכים).  
- **חיפוש** – בשדה החיפוש מעל הטבלה ניתן להקליד שם תלמיד.  
- **לחיצה על תלמיד** – נכנסים לפרופיל התלמיד.  

---

## פרופיל תלמיד

- **הערה כללית** – מוצגת בראש הפרופיל.  
- **הערכה חדשה** – טופס עם תאריך, ציון 1–100 לכל כישור (גרירה או מספר) והערה אופציונלית. שמירה מוסיפה הערכה ומעדכנת את התרשים.  
- **הערכה אחרונה** – כישורים ממוינים מהנמוך לגבוה עם ציונים.  
- **תרשים רדאר** – מציג את ההערכה הנבחרת.  
- **מצב השוואה** – סימון "השוואת הערכות" ובחירת שתי הערכות מציג שתי עקומות על גבי תרשים אחד והפרש מספרי בין ההערכות.  
- **תלמיד קודם / הבא** – מעבר בין תלמידים באותה קבוצה.  

---

## קבוצות

- **הוספת קבוצה** – להקליד שם בשדה "שם קבוצה" ולחץ "הוסף קבוצה".  
- **שינוי שם** – ללחוץ על כפתור העיפרון ליד שם הקבוצה, לערוך ולשמור.  
- **מחיקה** – ללחוץ פעם על אייקון המחיקה (לאשר), ולחוץ שוב למחוק.  

---

## ייצוא והדפסה

- **ייצוא CSV** – "ייצוא שלושת הקבצים" מוריד skills.csv, students.csv, assessments.csv.  
- **דוח קבוצה** (ממסך טבלת הקבוצה):  
  - **"העתק סיכום להדבקה ב-Google Docs"** – מעתיק לדף טקסט מוכן להדבקה.  
  - **"דוח להדפסה / PDF"** – פותח חלון עם טבלה; בדפדפן אפשר להדפיס או "שמירה כ-PDF".

- כדי לגבות או להעביר נתונים – השתמש ב**ייצוא CSV**.
